Interface IPipelineCharacterizer

  • All Known Implementing Classes:
    WEKAPipelineCharacterizer

    public interface IPipelineCharacterizer
    Finds patterns in given MLPipelines. A pipeline characterizer first has to be built with build(List), where it identifies patterns in the given data base of pipelines. Subsequently, it can be used to check for these patterns in a new pipeline.
