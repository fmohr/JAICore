    public interface ISolutionReportingNodeEvaluator<T,V extends java.lang.Comparable<V>>
extends INodeEvaluator<T,V>
    This interface suggests that the node evaluator MAY report solutions that he finds under a node. However, not every such solution NEEDS to be reported. In particular, algorithms using solution reports must anticipate that solutions are found that were not already reported by the node evaluator.
