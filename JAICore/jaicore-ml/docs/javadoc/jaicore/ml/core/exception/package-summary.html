Package jaicore.ml.core.exception Description

This package contains Exceptions defined by jaicore-ml. Most importantly it contains the two (abstract) base classes CheckedJaicoreMLException and UncheckedJaicoreMLException which can be used to catch all exceptions emitted by classes which are part of jaicore-ml.
